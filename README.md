Implementa un agente inteligente que resuelve el 8‑Puzzle utilizando el algoritmo A* con heurística de distancia Manhattan.
El agente recibe un estado inicial y encuentra la secuencia óptima de movimientos para llegar al estado objetivo.
El agente debe transformar un estado inicial como:

7 2 4

5 6 

8 3 1

En el estado objetivo:
  
  1 2
  
3 4 5

6 7 8

Utilizando la menor cantidad de movimientos posible.
¿Por qué es un agente inteligente?
Este proyecto cumple con los requisitos de un agente basado en objetivos:
- Percibe el estado actual del entorno (el tablero).
- Evalúa qué acciones son posibles.
- Predice el costo futuro mediante una heurística.
- Planifica una secuencia de acciones.
- Selecciona el camino más prometedor usando A*.
- Actúa racionalmente para alcanzar el objetivo.
